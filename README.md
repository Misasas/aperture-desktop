# Aperture Desktop

ローカルの画像・動画ファイルを管理するWindowsデスクトップアプリケーションです。

## 機能

- **Obsidian風サイドバー**: ツリー構造でフォルダを階層表示
- **グリッド表示**: サムネイル付きでファイル一覧を表示
- **ビューアー**: 画像・動画をフルスクリーンで表示
- **タグ機能**: フォルダ単位でタグを付与・検索
- **ファイル操作**: 新規フォルダ作成、名前変更、削除、ドラッグ&ドロップ
- **テーマ**: ライト/ダーク/クリーム

## セットアップ

### 必要な環境

- Node.js 20.x 以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

### ビルド

```bash
# 本番用ビルド
npm run build

# インストーラー作成
npm run dist
```

## 使い方

1. アプリを起動
2. 管理したいフォルダを選択
3. サイドバーでフォルダを選択、メインエリアで画像を表示
4. 右クリックでファイル操作、タグ編集が可能

## キーボードショートカット

| キー | 動作 |
|------|------|
| Backspace | 親フォルダに戻る |
| F5 | 表示を更新 |
| Ctrl+B | サイドバー表示/非表示 |
| ← → | ビューアーで前後移動 |
| Esc | ビューアーを閉じる |

## 技術スタック

- **Electron**: デスクトップアプリフレームワーク
- **React**: UIライブラリ
- **TypeScript**: 型安全な開発
- **Vite**: 高速ビルドツール
- **Sharp**: 画像処理（サムネイル生成）
- **Chokidar**: ファイル監視
